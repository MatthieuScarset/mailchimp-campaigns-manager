The best plugin to import Mailchimp campaigns into your WordPress site.

== Description ==

This plugin allows you to connect your blog with _Mailchimp Campaign Manager_ in order to import your newsletters in WordPress.

It's 100% free to use for standard users.

Major features include:

* Manually import Mailchimp campaigns in WordPress as custom post type
* Basic campaigns statistics
* Display your campaigns anywhere in WordPress with a shortcode

Paid features include:

* Automatic import/update of Mailchimp campaigns with WordPress
* Realtime campaigns statistics
* Export campaigns as HTML file
* Customize display in WordPress

PS: You'll be prompted [to register for free on this application](https://mailchimp-campaigns-manager.com) to be able to connect your WordPress site.

== Installation ==

Upload this plugin to your blog, activate it, and go to the settings page in WordPress.

1, 2, 3: You're done!
